# Proyecto Productos

Sistema de gestión de inventario de productos en Python con almacenamiento en base de datos SQLite.

## Características
- Registrar, visualizar, actualizar, eliminar y buscar productos.
- Reporte de productos con bajo stock.
- Separación en backend (lógica y base de datos) y frontend (menú e interacción).

## Requisitos
- Python 3.x

## Uso
1. Ejecuta `main.py`.
2. Sigue las instrucciones del menú para gestionar los productos.
